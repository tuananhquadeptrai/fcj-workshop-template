Tại đây sẽ là phần liệt kê, giới thiệu các blogs mà các bạn đã đăng trên [AWS Study Group](https://www.facebook.com/groups/awsstudygroupfcj). Ví dụ:

###  [Blog 1 - MCP không còn là tất cả trong kiến trúc Agentic AI](3.1-Blog1/)
Blog này tóm tắt ba giao thức quan trọng trong kiến trúc Agentic AI trên Amazon Bedrock AgentCore: MCP để kết nối Agent với công cụ, A2A để các Agent phối hợp với nhau, và AG-UI để cải thiện trải nghiệm tương tác giữa Agent và người dùng.

###  [Blog 2 - Các cách kiểm soát quyền truy cập ứng dụng Web trên AWS với Amazon Cognito](3.2-Blog2/)
Blog này trình bày ba mô hình kiểm soát quyền truy cập ứng dụng web bằng Amazon Cognito: tích hợp với Application Load Balancer, API Gateway và CloudFront kết hợp AWS Lambda. Bài viết so sánh cách hoạt động, ưu điểm, hạn chế và trường hợp sử dụng phù hợp của từng mô hình.

###  [Blog 3 - ...](3.3-Blog3/)
Blog này giới thiệu Amazon EKS Pod Identity vừa bổ sung tính năng session policies, cho phép bạn thu hẹp quyền IAM một cách linh hoạt và chính xác cho từng pod mà không cần tạo thêm nhiều IAM roles riêng biệt. Đây là bước tiến quan trọng giúp áp dụng nguyên tắc least privilege hiệu quả hơn trong môi trường Kubernetes quy mô lớn.
